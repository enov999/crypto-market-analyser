# Crypto Market Analyzer

## Overview
This project analyses historical cryptocurrency price data to identify trends, volatility, and correlations between major assets.

## Motivation
I built this project to develop my Python programming skills and gain a deeper understanding of how financial market data can be analysed using code.

## Features
- Fetches historical cryptocurrency price data from a public API
- Calculates moving averages and volatility
- Visualises price trends using charts

## Technologies Used
- Python
- pandas
- matplotlib
- requests

## Future Improvements
- Add support for more cryptocurrencies
- Improve performance for larger datasets
- Add basic automated test
